# blog-app-backend
It is a backend of blog application developed in spring boot and hibernate. Basically this project has the API's of blog app which are secured 
using spring security and it has JWT based authentication to role wise permit the users to access the API's. This project covers many more API
development concepts in java spring boot as below:

Key concepts you'll get to learn from the project:
1. API developments using spring boot with HTTP requests (GET, POST, PUT, DELETE)
2. Hibernate ORM
3. Bean Validation
4. Spring Security
5. JWT Authentication
6. Exception Handling at global level in project.
7. Entity parsing using ModelMapper
8. Lombok Annotations
